# 📊 Data Analytics Project

## 🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data to actionable insights. It includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and final reporting. The goal is to showcase practical data analytics skills using industry-standard tools.

---

## 📁 Dataset

* Source: Customer Shopping Behavior Dataset
* Format: CSV
* Size: ~3900 records
* Key Columns:

  * Customer ID
  * Age
  * Gender
  * Item Purchased
  * Category
  * Purchase Amount
  * Review Rating
  * Discount Applied
  * Frequency of Purchases

## ⚙️ Project Workflow

### 1️⃣ Data Loading (Python)

* Imported dataset using Pandas
* Verified structure using `.info()` and `.head()`

### 2️⃣ Data Cleaning

* Handled missing values (e.g., Review Ratings using median)
* Standardized categorical values
* Converted data types where necessary

### 3️⃣ Exploratory Data Analysis (EDA)

* Analyzed distributions (age, purchase amount)
* Identified trends across categories and seasons
* Checked relationships between variables

### 4️⃣ SQL Analysis

* Imported cleaned dataset into SQL
* Performed analytical queries such as:

  * Top products by sales
  * Discount impact analysis
  * Customer purchase patterns
  * Category-wise performance

### 5️⃣ Power BI Dashboard

* Built interactive dashboard with:

  * Sales overview
  * Category performance
  * Customer segmentation
  * Discount insights
* Used filters and slicers for dynamic analysis

### 6️⃣ Reporting

* Created a structured report summarizing:

  * Key insights
  * Business recommendations
  * Data-driven conclusions

### 7️⃣ Presentation (Gamma)

* Designed a clean and professional presentation
* Highlighted:

  * Problem statement
  * Approach
  * Insights
  * Visuals from dashboard
